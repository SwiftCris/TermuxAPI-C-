# TermuxAPI-C-

TermuxAPI Wrapper for C++

# How it works
This was written in C++ and it uses popen and pipes to execute termux api commands.

# How to use?
Just download the latest .hpp file from the resources folder, or grab it and then, include it in your Project!

# Documentation 
As of now, the available methods are:

TermuxAPI.flashlightOn()
TermuxAPI.flashlightOff()
TermuxAPI.termuxElfCleaner()
and TermuxAPI.getBatteryStatus()
